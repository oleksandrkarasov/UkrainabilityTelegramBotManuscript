# Code for the manuscript 'Mapping quality time: Telegram as a public participatory GIS for cultural ecosystem services'.

'Notebook_for_figures_and_tables.ipynb' contains the code used to produce Figures 1, 5-7, and Tables 1-3, as well as Supplementary Materials.

'english_data_for_sharing.xlsx' contains anonymised, translated, non-spatial dataset, used to conduct the analysis.

'association_tests_multi_select.csv', 'association_tests_single_select.csv' are the outputs of the statistical analysis (Tables 1,2). 

'model_comparison_report.xlsx' contains comparison of Random Forest and CatBoost modelling approaches.
